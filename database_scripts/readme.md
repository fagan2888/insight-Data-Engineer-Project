# Instruction to use the database scripts

These scripts build the database from the all the Freddie Mac and Fannie Mae data sets on your s3 and get you started on some of the cool queries you can realize now that you have all the pieces at your disposal.

Step 1: run the raw schema files. The raw schema files specify how all the columns should be read and specify the sortkeys and distribution keys for the tables.

Step 2: run the populate files. The populate files will populate the raw tables from all the raw text files we just uploaded to S3.
